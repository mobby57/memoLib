## 🚀 DÉMARRAGE RAPIDE (5 minutes)

### Étape 1: Inscription (2 min)

#### Accéder à l'application
```
URL: http://localhost:5078
ou
URL: https://app.memolib.com
```

#### S'inscrire
1. Cliquez sur **"S'inscrire"**
2. Remplissez le formulaire:
   ```
   Email: votre-email@cabinet.fr
   Mot de passe: (minimum 8 caractères)
   Nom: Maître [Votre Nom]
   Cabinet: [Nom de votre cabinet]
   Téléphone: +33 X XX XX XX XX
   ```
3. Cliquez sur **"Créer mon compte"**

✅ **Vous êtes inscrit !**

---

### Étape 2: Première Connexion (1 min)

1. Connectez-vous avec vos identifiants
2. Vous arrivez sur le **Dashboard**

**Ce que vous voyez:**
- 📊 Statistiques (vides pour l'instant)
- 📧 Emails récents
- 📁 Dossiers en cours
- 🔔 Notifications

---

### Étape 3: Configuration Email (2 min)

#### Option A: Gmail (Recommandé)

1. Allez dans **Paramètres** > **Email**
2. Cliquez sur **"Connecter Gmail"**
3. Suivez les étapes:
   ```
   1. Activez la validation en 2 étapes sur Gmail
   2. Créez un mot de passe d'application:
      https://myaccount.google.com/apppasswords
   3. Copiez le mot de passe généré
   4. Collez-le dans MemoLib
   ```
4. Cliquez sur **"Tester la connexion"**

✅ **Gmail connecté !**

#### Option B: Autre Email

1. Allez dans **Paramètres** > **Email**
2. Entrez vos paramètres IMAP:
   ```
   Serveur IMAP: imap.votre-fournisseur.com
   Port: 993
   Email: votre-email@cabinet.fr
   Mot de passe: [votre mot de passe]
   ```
3. Cliquez sur **"Tester la connexion"**

## 📁 CRÉER VOTRE PREMIER DOSSIER (5 minutes)

### Depuis un Email

1. Allez dans **Emails**
2. Cliquez sur un email reçu
3. Cliquez sur **"Créer un dossier"**
4. Le formulaire est pré-rempli:
   ```
   ✅ Titre: [Extrait du sujet]
   ✅ Client: [Nom extrait]
   ✅ Email: [Email extrait]
   ✅ Téléphone: [Téléphone extrait]
   ```
5. Complétez si nécessaire
6. Cliquez sur **"Créer"**

✅ **Dossier créé !**

---

### Manuellement

1. Allez dans **Dossiers**
2. Cliquez sur **"+ Nouveau dossier"**
3. Remplissez:
   ```
   Titre: Contentieux commercial - Jean Dupont
   Client: Jean Dupont
   Email: jean.dupont@email.com
   Téléphone: 06 12 34 56 78
   Statut: OUVERT
   Priorité: 3 (Normal)
   ```
4. Cliquez sur **"Créer"**

✅ **Dossier créé !**

## 💡 ASTUCES PREMIER CLIENT

### 1. Importez vos Emails Existants

**Méthode:**
1. Allez dans **Paramètres** > **Import**
2. Cliquez sur **"Scanner les emails existants"**
3. Sélectionnez la période (ex: 30 derniers jours)
4. Cliquez sur **"Importer"**

**Résultat:**
- Tous vos emails récents dans MemoLib
- Clients détectés automatiquement
- Dossiers suggérés

---

### 2. Créez vos Templates

**Templates utiles:**

**Template 1: Confirmation RDV**
```
Objet: Confirmation de rendez-vous - {{clientName}}

Bonjour {{clientName}},

Votre rendez-vous est confirmé pour le {{date}} à {{heure}}.

Adresse: {{cabinetAddress}}

Merci de vous munir de vos pièces d'identité.

Cordialement,
{{lawyerName}}
```

**Template 2: Demande de Pièces**
```
Objet: Dossier {{caseTitle}} - Pièces manquantes

Bonjour {{clientName}},

Pour avancer sur votre dossier, j'ai besoin des pièces suivantes:
- {{piece1}}
- {{piece2}}
- {{piece3}}

Merci de me les transmettre avant le {{deadline}}.

Cordialement,
{{lawyerName}}
```
