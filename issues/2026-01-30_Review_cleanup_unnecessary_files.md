Title: Review and cleanup unnecessary files

This issue is created to track and manage unnecessary files in the repository.
Criteria for removal:

1. Temporary files (e.g., .tmp, .log, .cache)
2. Cached files (e.g., __pycache__/ and other compiled files)
3. Example or skeleton configuration files that are not used (e.g., .env.example).

Please review all files and confirm before proceeding with deletions.
