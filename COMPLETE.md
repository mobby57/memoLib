## ⚙️ CONFIGURATION OPTIONNELLE

### Gmail (Monitoring automatique)

Pour activer le monitoring automatique des emails:

```powershell
# 1. Créer mot de passe app Gmail
# https://myaccount.google.com/apppasswords

# 2. Configurer
.\FIX-ALL.ps1 -GmailPassword "votre-mot-de-passe-app"

# 3. Redémarrer
.\START.bat
```

### Déploiement Cloud (Fly.io)

Pour déployer en production cloud:

```powershell
# 1. Initialiser
.\DEPLOY-FLY.ps1 -Init

# 2. Suivre les instructions

# 3. Déployer
.\DEPLOY-FLY.ps1
```
