## 🐛 DÉPANNAGE

### Scripts ne s'activent pas
1. Vérifier Tampermonkey installé
2. Dashboard → Vérifier scripts activés
3. Rafraîchir la page (F5)
4. Vérifier URL match (@match)

### Conflits entre scripts
- Désactiver `7-modern-ui.user.js` si `9-ultimate-modern-ui.user.js` actif
- Un seul script UI à la fois

### Performance lente
- Désactiver `8-advanced-components.user.js` (Chart.js)
- Désactiver `5-ocr-helper.user.js` (Tesseract.js)
